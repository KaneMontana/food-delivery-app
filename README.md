# food-delivery-app
Приложение для доставки еды
# Приложение позволяет пользователям:
- просматривать каталог ресторанов и меню
- искать блюда по категориям
- добавлять позиции в корзину
- оформлять и оплачивать заказ
- отслеживать статус доставки в реальном времени
# Структура репозитория
- MobileApp
- ServerApi
- Web
